# Fundo Mágico — Gerador de Background com IA

O **Fundo Mágico** é uma aplicação web que permite ao usuário criar backgrounds personalizados a partir da combinação de cores, utilizando Inteligência Artificial.  
Além da visualização em tempo real, o projeto também disponibiliza os códigos HTML e CSS prontos para reutilização.

## Funcionalidades

- Seleção e combinação de cores pelo usuário  
- Geração automática de backgrounds com IA (Gemini)  
- Visualização do resultado em tempo real  
- Exibição dos códigos HTML e CSS correspondentes  
- Integração com n8n para automação do fluxo  
- Interface responsiva  

## Tecnologias Utilizadas

- HTML5 — Estrutura da aplicação  
- CSS3 — Estilização e layout  
- JavaScript (Vanilla) — Lógica e interações  
- n8n — Orquestração e automação do fluxo  
- Gemini (Google AI) — Geração inteligente dos backgrounds  

## Como Funciona

1. O usuário escolhe uma ou mais cores na interface  
2. As informações são enviadas via JavaScript para um fluxo no n8n  
3. A IA (Gemini) processa as cores e gera um background estilizado  
4. O resultado é retornado para a aplicação  
5. A página exibe:
   - O background gerado em tempo real  
   - O código HTML e CSS correspondente
